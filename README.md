## ⚙️ Uso

1. Abrir la web
2. Al inicio se pregunta el total del dinero disponible.
3. Posteriormente colocar el nombre del gasto y el costo.
4. Luego se ira descontando del total del dinero disponible
5. EL proyecto cuenta con un tope de saldo negativo disponible
